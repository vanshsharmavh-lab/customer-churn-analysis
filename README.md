# 📊 Customer Churn Analysis

## 📌 Overview

Customer churn is one of the biggest challenges for subscription-based businesses. This project analyzes customer data from a telecom company to identify patterns and factors associated with customer churn. Using Python, the dataset was cleaned, explored, and visualized to generate actionable business insights that can support customer retention strategies.

---

## 🎯 Objectives

- Clean and prepare the customer dataset
- Perform exploratory data analysis (EDA)
- Analyze customer churn patterns
- Identify factors influencing churn
- Generate business insights using visualizations

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib

---

## 📂 Dataset

The project uses the **IBM Telco Customer Churn** dataset containing customer demographics, account information, subscribed services, and churn status.

Key columns include:

- Gender
- Senior Citizen
- Contract Type
- Payment Method
- Internet Service
- Monthly Charges
- Total Charges
- Tenure
- Churn

## 💡 Key Insights

- Calculated the overall customer churn rate.
- Compared churn across different contract types.
- Analyzed payment methods associated with higher churn.
- Examined the relationship between monthly charges and customer churn.
- Compared average customer tenure between retained and churned customers.
